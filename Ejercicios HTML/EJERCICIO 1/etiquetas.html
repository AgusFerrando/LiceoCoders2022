INTRODUCCION A HTML

Todas las etiquetas en HTML se deben incluir dentro de los símbolos "<" y ">", agregando una barra "/" a la etiqueta de cierre. Así una etiqueta por ejemplo de parrafo quedara de la siguiente manera:
<p> </p>

Listado de etiquetas

  • strong pone el texto en negrita

  • ins subraya el texto

  • em pone el texto en cursiva

  • small encoge el texto

  • big agranda el texto

  • subsub añade un subindice

  • sup sup añade un superindice

  • strike tacha el texto

  • pre indica a los navegadores que deben respetar los saltos de línea y los espacios en blanco de un documento pero no respeta los caracteres especiales como <

  • abbr representa una abreviación y se utiliza asi:
    <abbr title="Hypertext Markup Language"> HTML </abbr>

  • mark resalta el texto

Una etiqueta muy interesante para mostrar código preformateado es code.
Con ella podremos mostrar el codigo como se ve en el editor o de la siguiente manera: <code> <h1> Título principal </h1> <code/>
