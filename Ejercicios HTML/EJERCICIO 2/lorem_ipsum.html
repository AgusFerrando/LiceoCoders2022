INTRUDUCCION A HTML

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos 💻.

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido.

Consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.

Si quieres especificar que se trata de un párrafo, podrías encerrar el texto con la etiqueta de párrafo <p> </p>

Por ejemplo <p> Mi gato es muy gruñon </p>

Las partes principales de un párrfo son 👇

  • ETIQUETA DE APERTURA: consiste en el nombre del elemento (en este caso, p), encerrado por paréntesis angulares (< >) de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento

  • CONTENIDO: este es el contenido del elemento, que en este caso es sólo texto.

  • ELEMENTO: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.

  • ETIQUETA DE CIERRE:es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento.

HTML es el primer lenguaje que debe aprender cualquier persona interesada en construir un sitio web.

A partir del mismo podemos pasar a muchos otros lenguajes interesantes que sirven para hacer cosas diversas y más avanzadas.

Aprender HTML es sencillo. En pocos días o semanas serás capaz de entender y usar las etiquetas más comunes y componer documentos HTML (páginas web) correctas 😎 .
